A modern Android application that allows users to browse and explore sneaker collections through a clean and intuitive user interface. The project demonstrates Android application development concepts, responsive layouts, and efficient navigation patterns.

---

## 📱 Overview

Sneakers App is designed to provide users with an engaging shopping-style experience where they can discover sneakers, view product details, and navigate seamlessly through the application.

### ✨ Features

* Browse sneaker collections
* View sneaker details
* Clean and modern UI
* Smooth screen navigation
* Responsive Android layouts
* User-friendly experience

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/rohit220604/SneakerApp.git
```

### Open in Android Studio

1. Open Android Studio
2. Select **Open Project**
3. Choose the cloned repository
4. Sync Gradle dependencies
5. Run the application on an emulator or physical Android device
